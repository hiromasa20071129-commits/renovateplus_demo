## 💡 工夫したポイント  

- HTML/CSSで全ページ構築（トップ・サービス・施工事例・お問い合わせ・サンクスページ）  
- Formspreeを使ってお問い合わせフォームを実装  
- 統一感のあるデザイン（カラー・余白・フォント）  
- レスポンシブ対応でスマホでも見やすい設計  
- GitHub Pagesで無料公開し、学習成果を見える形に
